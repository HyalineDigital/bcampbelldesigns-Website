# Google Sans Flex Font Files

Place the Google Sans Flex font files in this directory with the following naming convention:

- `GoogleSansFlex-Regular.woff2` (400 weight)
- `GoogleSansFlex-Medium.woff2` (500 weight)
- `GoogleSansFlex-SemiBold.woff2` (600 weight)
- `GoogleSansFlex-Bold.woff2` (700 weight)
- `GoogleSansFlex-ExtraBold.woff2` (800 weight)
- `GoogleSansFlex-Black.woff2` (900 weight)

## How to Get Google Sans Flex:

Google Sans Flex is available under the SIL Open Font License. You can download it from:
- Google Fonts: https://fonts.google.com/noto/specimen/Noto+Sans
- Or search for "Google Sans Flex download" for direct links

The font files should be in WOFF2 format for best browser support and performance.
